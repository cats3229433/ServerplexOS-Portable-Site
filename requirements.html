System Requirements

Everything you need to run ServerplexOS Next Portable. Requirements are minimal — if you have a browser and a code editor, you're almost certainly good to go.

// hardware & software specs
Component Minimum Recommended
Browser Any modern browser (Chromium, Firefox, Safari) Chrome / Chromium 110+
Code Editor Any editor that can open .html files VS Code (latest)
RAM 2 GB available 4 GB or more
Storage Enough to save a single HTML file USB 3.0 drive or SSD
Internet Not required to run Required for App Store & Browser
JavaScript Must be enabled in browser Enabled (default on all browsers)
